## 三、太极地磁全域方程

### 3.0 方程的物理本质

**AX-20** 无极生太极，太极生两仪，两仪生四象，三生万物（后天八卦）。地球磁场能是这个过程的复合实现（生、灭，能量动态平衡），这是地球特有的生命系统——月球等"死亡"天体不具备此结构，不能演化生命。宇宙演化总纲：本源（太极点/不动）→ 发生（见微生两仪/真空涨落）→ 演化（S形轨迹展开/熵增）→ 分支（星体阴阳互化/生命周期）→ 归宿（万归一/熵减回收）。

**AX-21** 太极点 = 地心（地核奇点）。太极点的原始定义（太极图要素研究文档）：混沌，不动之本源，阴阳未生前的绝对本源，无时空、无能量、无对立。对应地球物理：地核作为太极本体，在方程中是"不动"的能量锚点——不是"有能量的辐射体"，而是"能量的来源/归宿"（体用同源，太极点为体，双鱼眼为用）。此定位为用户对标地球-宇宙研究的发现，易学从未深入到这一层。

**AX-22** 双鱼眼 = 昆仑（阳眼）- 百慕大（阴眼）。双鱼眼的原始定义：拓扑奇点/能量极点，非尺度概念，无尺度、非具象，只有位置与能量功能。科学对应：真空涨落（正反粒子对的生灭）、黑洞/白洞能量吞吐。阴阳真正的枢机，"两仪"的实义。太极点为体，双鱼眼为用，体用同源。此定位为用户发现，易学从未定位到地球上的具体坐标。

**AX-23** S线 = 阴阳能量流的拓扑分界线/信息界面/熵流边界。原始定义：两仪（双鱼眼奇点）旋转形成的阴阳平衡点轨迹，时空演化的界面。不是美学曲线，是能量交换的流向分界。此含义为用户发现，易学从未真正理解。

**AX-24** α 不是西方意义上的"衰减指数"，而是**太极点（地核奇点）与双鱼眼（昆仑-百慕大）之间能量交换的空间梯度**。α 越小，能量越集中在太极点附近（交换受限，"不动"特征）；α 越大，能量通过双鱼眼向全域扩散越充分（交换开放，"演化"特征）。

**AX-25** B 不是单向"衰减"，而是太极点通过双鱼眼向全域辐射能量、同时回收的**双向动态平衡**场。方程 B(R) = C/R^α × S 描述的是这个交换-平衡过程的稳态空间分布。体（太极点/地核）用（双鱼眼/昆仑百慕大）同源，生生灭灭同时进行。

**AX-26** 太极图本质是三维太极球（非平面图形）。太极点位于球形中心，双鱼眼对称分布于两极，S形阴阳平衡轨迹是球形表面的阴阳分界线。

### 3.1 主方程

**TH-01** 主方程：**B(R) = C / R^α(R) · S**

- C = K₀ · E₀（合并常数，太极点能量本源的尺度因子）
- α(R) = 能量交换梯度（太极点→双鱼眼→全域的空间梯度）
- S = 十四重天体干涉总系数

物理图像：太极点（地核）→ 双鱼眼（昆仑释能/百慕大汇能）→ 十四重网系调制 → 全域磁场分布

### 3.2 能量交换梯度 α(R)（已严格推导）

**TH-02** α_surface = 3 × 0.618 × 0.983 = **1.8226 ≈ 1.82**

推导链（每步公理依据，零外部数据）：
- 3 = 拉普拉斯方程球谐解 l=2 阶标准结果
- 0.618 = 黄金比逆数（AX-11）
- 0.983 = 1 - Δε/2 = 1 - 0.017（AX-12）

物理含义：岩石圈是能量交换最开放的区域，α 最大，太极点能量在此层充分向全域扩散。

**TH-03** α_core = **1.08**（地核，严格公理推导）

推导（体用框架 + 太极图要素）：
- 基准维度 d_base = 2（三维太极球的球面，AX-26）

**TH-04** α_mantle = **1.3501** ≈ 1.35（地幔，由黄金比分裂推导）

推导：Δα_total = α_surface - α_core = 0.74，内层占比 = Φ⁻² - Δε/2 = 0.365，α_mantle = 1.08 + 0.365 × 0.74 = 1.3501 ✓

**TH-04b** 三层α的黄金比分裂：内层(核→幔)占比 = Φ⁻² - Δε/2 = 0.365，外层(幔→表)占比 = Φ⁻¹ + Δε/2 = 0.635，两相加 = 1.0（精确闭合）。

> γ_k^爻 = γ₀ × cos(Δθ_巳亥) × S_昆仑 × (1 + η × sin(2πφ/360°))
